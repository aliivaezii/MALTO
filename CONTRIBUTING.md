# Contributing

Thank you for your interest in this project.

This repository was created as a solo entry for the MALTO Recruitment Hackathon.
Since it is a competition submission, contributions are not accepted during the
competition period.

After the competition ends, feel free to open an issue or submit a pull request
if you spot bugs, have ideas for improvement, or want to extend the pipeline.

## How to Contribute

1. Fork the repository
2. Create a feature branch (`git checkout -b feature/your-feature`)
3. Commit your changes (`git commit -m "Add your feature"`)
4. Push to the branch (`git push origin feature/your-feature`)
5. Open a Pull Request

## Code Style

- Follow PEP 8 for Python code
- Use type hints where reasonable
- Keep notebooks clean: restart kernel and run all before committing
- Write clear docstrings for functions in `src/`
